# fish config

use the `fish_config` command to edit colors and prompt.

### install

```shell
mv ~/.config/fish/config.fish ~/.config/fish/config.fish.bk
ln -s .../dotfiles/config.fish ~/.config/fish/config.fish
mv ~/.config/fish/conf.d ~/.config/fish/conf.d.bk
ln -s .../dotfiles/fish/ ~/.config/fish/conf.d
```
